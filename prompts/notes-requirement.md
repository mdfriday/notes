## **📌 整体页面结构**

## **🚀 设计优化点**
✅ **简洁 & 轻量**：侧边栏可折叠，Modal 处理短时任务。  
✅ **模块化布局**：各功能模块按需展开，避免干扰创作。  
✅ **高效交互**：文件 & 资源管理方便，Markdown 编辑体验流畅。  
✅ **一致性**：所有交互符合 Markdown 编辑直觉，提高用户习惯适配性。  

---

### **📌 设计总结**
- **侧边栏（可折叠）** → 适用于长期管理：项目、文件、资源。
- **Modal（弹窗）** → 适用于短时任务：创建项目、选择图片。
- **核心编辑体验保持不变**，增强 Markdown & 资源管理的便利性。


## 技术要求

1. 使用 TypeScript 和 React 开发, 使用 Vite 作为构建工具
2. 样式使用 Tailwind CSS
3. Icon 作用 Tailwind 的 svg 图标
